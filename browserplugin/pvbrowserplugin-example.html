This example demonstrates the combination of Javascript and a pvbrowser plugin.

Linux installation:

# for 64 bit system
cp /opt/pvb/browserplugin/libpvbrowser.so /usr/lib64/browser-plugins/ 
# for 32 bit system
cp /opt/pvb/browserplugin/libpvbrowser.so /usr/lib/browser-plugins/

Windows installation:

cd "c:\program files\mozilla firefox\plugins"
copy "%PVBDIR%\win-mingw\bin\*.dll"
